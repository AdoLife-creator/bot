# PvPToggle Plugin

Paper 1.21.4-1.21.8 sunucuları için PvP Toggle Plugin

## Özellikler

- ✅ `/pvp` komutu ile PvP'yi açıp kapatabilme
- ✅ Oyuncu PvP kapalıyken hasar almaz ve veremez
- ✅ Türkçe mesajlar
- ✅ Permission sistemi (pvptoggle.use)

## Kurulum

1. `PvPToggle-1.0.0.jar` dosyasını `plugins` klasörüne koyun
2. Sunucuyu yeniden başlatın
3. `/pvp` komutu ile kullanmaya başlayın

## Komutlar

- `/pvp` - PvP'yi aç/kapat

## İzinler

- `pvptoggle.use` - /pvp komutunu kullanabilme (varsayılan: true)
